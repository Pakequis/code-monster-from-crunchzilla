Como brincar

Code Monster ensina crianças e adultos um pouco sobre programação Javascript!

Sobre

Code Monster da Crunchzilla é um tutorial interativo para crianças que foca na ação. Mudanças no código imediatamente levam a resultados visíveis.

Perguntas mais comuns

O que é esse c em coisas como c.save()?

É um pouco técnico mas, se você quiser saber mais detalhes, c é uma variável que o Code Monster cria para você e que contém o contexto de tela 2D. O código que cria a variável c, que fica escondido, parece com isso

var c = document.getElementById('pane').getContext('2d');

Como eu salvo meu código e uso fora do Code Monster?

Resposta curta: Não é super fácil, mas você pode copiar e colar seu código e carregar seu código em um arquivo HTML (carregando um arquivo Javascript separado ou inserindo o Javascript no seu arquivo HTML)

Resposta comprida: Um arquivo HTML minimo seria mais ou menos assim:

<html><body>
<canvas id=pane width=400 height=300></canvas>
<script>
var c = document.getElementById('pane').getContext('2d');

// COLE SEU CÓDIGO AQUI

</script></body></html>

Se você copiar isso aí para um arquivo, depois copiar e colar seu código nele, depois salvar o arquivo com um nome terminado em .html você poderá abri-lo em qualquer navegador.

Existe mais algum código escondido de mim?

Existe um pouquinho de código escondido que tenta parar as animações quando você troca de lição e o seu código é preparado para tentar prevenir loops infinitos. Você não precisa desses para rodar seu código fora do Code Monster.

Por que você não me conta mais sobre a sintaxe e terminologia do Javascript?

Já existem muitos tutoriais e livros que ensinam programação Javascript explicando o que é uma variável, o que é escopo da variável, como a sintaxe do if se parece, como cada operador funciona e por aí vai. Se você que isso existem muitas opções disponíveis, muitas que são excelentes suplementos para o Code Monster.

O que é difícil de encontrar é um tutorial divertido, imersivo e que leva você a construir coisas legais imediatamente. Code Monster foge muito da sintaxe e das explicações que podem ser encontradas em outros lugares (Use o Google quando precisar) e pula direto para a diversão do que você pode fazer se aprender a programar.

Por que não funciona direito no Ipad (ou outro dispositivo móvel)?

Code Monster vai carregar em muitos dispositivos móveis, pode até ser um pouco jogável dependendo do seu dispositivo, mas não foi feito para isso. Por que não?

Muito por causa de ser difícil digitar nesses dispositivos. Escrever grande textos em um dispositivo móvel é difícil e é ainda pior quando o que você quer escrever é um código e precisa ficar olhando em outras partes do código. Tentar escrever grande quantidade de código tradicional não é uma boa experiência em dispositivos móveis.

Em parte por causa disso parece que ninguém tenha criado um grande aplicativo para aprender a programar, mas existem bons aplicativos que ensinam conceitos de programação (como o Hopscotch, Move the Turtle e Cargo-bot).

Posso usar o Code Monster Offline (quando não conectado a Internet)?

Claro! Code Monster só usa a internet bem no começo, para carregar o site, depois ele nunca mais usa a Internet.

Então, por exemplo, você pode carregar o Code Monster num laptop e depois pegar um avião ou andar no parque sem conexão a internet, abrir seu laptop lá e usar o Code Monster sem Internet. Ele até salva onde você está, assim você pode voltar para a mesma lição que estava! Você não precisa de uma conexão a internet exceto no começo quando a página carrega.
